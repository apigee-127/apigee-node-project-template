## Installation

swagger-core-api is distributed via [NPM][npm] so installation is the usual: `npm install swagger-core-api --save`.

## Contributing

This project uses [Gulp][gulp] for building so `npm install -g gulp` once you clone this project.  Running `gulp` in the
project root will lint check the source code and run the unit tests.
